#  3D Interactive Stands Map

An interactive 3D exhibition hall map built with Next.js 14, React Three Fiber, and TypeScript.

##  Features

-  Beautiful Modern UI with glass-morphism design
-  Fully Responsive - Auto-detects device type
-  Interactive 3D Stands with hover effects
-  Intuitive Controls optimized for each device
-  Fast & Optimized with Next.js 14

##  Quick Start

\\\ash
npm install
npm run dev
\\\

Open http://localhost:3000

##  Deploy

\\\ash
npm run build
npm start
